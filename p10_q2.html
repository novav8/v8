Practical Slip 10 - Question 2

Q2. Create a Wireless Network of Two or More PCs Using an Access Point

Aim:
To create a wireless network using an Access Point in Cisco Packet Tracer.

Procedure:
1. Place an Access Point and two or more PCs in Packet Tracer.
2. Configure SSID in the Access Point.
3. (Optional) Set wireless security password.
4. Connect PCs to the wireless network.
5. Assign IP addresses (Static or DHCP).
6. Test connectivity using ping command.

Result:
The wireless network was successfully created and connectivity was verified.
