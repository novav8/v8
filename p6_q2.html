Practical Slip 6 - Question 2

Q2. Verify IP Allocation

Aim:
To verify that the client PC has received IP address from the DHCP server.

Procedure:

1. Click on PC → Desktop → Command Prompt.
2. Type the following command:
   ipconfig
3. Check the assigned IP Address, Subnet Mask and Default Gateway.
4. To test network connectivity, type:
   ping 192.168.1.1
5. If replies are received, the network connection is successful.

Result:
The IP address is successfully assigned by the DHCP server and connectivity is verified.
