Aim:

To study the working of Ultrasonic sensors using Arduino.

Simulation Environment:

TinkerCAD (Free online simulator)

Components:

Arduino UNO
HC-SR04 Ultrasonic Sensor

Theory:

The HC-SR04 is an inexpensive and widely used ultrasonic distance sensor module. It is commonly used in robotics, automation, and DIY electronics projects.

The HC-SR04 sensor utilizes ultrasonic sound waves to determine the distance between the sensor and an object.

  • Ultrasonic Emission: The sensor emits a high-frequency sound wave (around 40 kHz) which is inaudible to humans.
  • Sound Wave Reflection: The emitted sound wave travels through air until it hits an object and bounces back toward the sensor.
  • Receiving the Echo: The sensor has a built-in receiver to detect the reflected sound wave (echo).
  • Calculating Distance: By measuring the time taken for the sound wave to travel to the object and return, the sensor calculates the distance using the speed of sound (approximately 343 m/s at room temperature).
  • Output: The calculated distance is provided as a digital pulse duration in microseconds, which can be converted into centimeters or inches.
  • This distance measuring technique is non-contact and suitable for applications like obstacle detection, liquid level measurement, and robotics.

Conclusion:

The HC-SR04 ultrasonic sensor successfully measures distance using ultrasonic waves and can be easily interfaced with Arduino for various real-world applications. 


// Define the pins for the ultrasonic sensor
const int trigPin = 9;   // Arduino digital pin for the trigger
const int echoPin = 10;  // Arduino digital pin for the echo

// Variables to store the duration and distance
long duration;
int distance;

void setup() {
    // Initialize serial communication for debugging
    Serial.begin(9600);

    // Define the trigger and echo pins as OUTPUT and INPUT
    pinMode(trigPin, OUTPUT);
    pinMode(echoPin, INPUT);
}

void loop() {
    // Trigger a pulse to the sensor
    digitalWrite(trigPin, LOW);
    delayMicroseconds(2);
    digitalWrite(trigPin, HIGH);
    delayMicroseconds(10);
    digitalWrite(trigPin, LOW);

    // Measure the duration of the pulse from the echo
    duration = pulseIn(echoPin, HIGH);

    // Calculate the distance based on the speed of sound
    distance = duration * 0.034 / 2; // Divide by 2 because sound travels to object and back

    // Print the distance to the serial monitor
    Serial.print("Distance: ");
    Serial.print(distance);
    Serial.println(" cm");

    // Add a delay between measurements
    delay(1000); // 1 second
}
