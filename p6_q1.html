Q1. Configure a DHCP Server Network and Connect a Client PC

Aim:
To configure a DHCP server and assign IP address automatically to a client PC.

Requirements:
1 Server
1 Switch
1 PC
Straight-through cables

Procedure:

1. Place Server, Switch and PC in Packet Tracer.
2. Connect Server to Switch using straight-through cable.
3. Connect PC to Switch using straight-through cable.
4. Click on Server → Desktop → IP Configuration.
   IP Address: 192.168.1.1
   Subnet Mask: 255.255.255.0
5. Go to Server → Services → DHCP.
   Turn DHCP Service: ON
   Pool Name: LAN
   Default Gateway: 192.168.1.1
   DNS Server: 8.8.8.8
   Start IP Address: 192.168.1.10
   Subnet Mask: 255.255.255.0
   Click Add
6. Click on PC → Desktop → IP Configuration.
   Select DHCP.

Result:
The client PC receives IP address automatically from the DHCP server.
