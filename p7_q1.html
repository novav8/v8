Q1. Create a Network of 5 PCs Using DHCP

Aim:
To create a network of 5 PCs using DHCP in Cisco Packet Tracer.

Requirements:
1 Server (DHCP enabled)
1 Switch
5 PCs
Straight-through cables

Procedure:
1. Place one Server, one Switch, and five PCs in Packet Tracer.
2. Connect all PCs and Server to the Switch using straight-through cables.
3. Configure Server with a static IP address (e.g., 192.168.1.1).
4. Go to Server → Services → DHCP and turn DHCP service ON.
5. Create a DHCP pool (set default gateway, subnet mask, start IP address).
6. On each PC, go to Desktop → IP Configuration and select DHCP.
7. Verify that all PCs receive IP addresses automatically.

Result:
All 5 PCs successfully received IP addresses using DHCP.
