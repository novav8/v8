Practical Slip 10 - Question 1

Q1. Verify IP Allocation Using: ipconfig or ifconfig

Aim:
To verify IP address allocation using ipconfig (Windows) or ifconfig (Linux).

Procedure:
1. Open Command Prompt (Windows) or Terminal (Linux).
2. Type the command:
   ipconfig (for Windows)
   ifconfig (for Linux)
3. Press Enter.
4. Check the displayed IP Address, Subnet Mask, and Default Gateway.

Result:
The IP configuration details were successfully displayed and verified.
