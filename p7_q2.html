Practical Slip 7 - Question 2

Q2. Display and Write IP Address, Subnet Mask and Default Gateway of Any Two PCs

Aim:
To display IP configuration details of any two PCs.

Procedure:
1. Open any PC → Desktop → Command Prompt.
2. Type: ipconfig
3. Note the IP Address, Subnet Mask, and Default Gateway.
4. Repeat the same steps for another PC.

Result:
IP Address, Subnet Mask, and Default Gateway of two PCs were successfully displayed.
