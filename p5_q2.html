Practical Slip 5 - Question 2

Q2. Verify Connectivity Using Ping Command and Note the Output

Aim:
To verify connectivity between the PCs using ping command.

Procedure:
1. Open Command Prompt on any PC.
2. Type: ping <IP address of another PC>
3. Press Enter.
4. Check for "Reply from" messages.

Result:
Successful reply messages confirm that the connectivity between PCs is working properly.
