Q1. Create a Wireless Network of Two or More PCs Using an Access Point

Aim:
To create a wireless network of two or more PCs using an Access Point in Cisco Packet Tracer.

Requirements:
1 Access Point
2 or more PCs with wireless module

Procedure:
1. Place an Access Point and two or more PCs in Packet Tracer.
2. Configure SSID in the Access Point.
3. (Optional) Set wireless security such as WPA2 password.
4. On each PC, go to Desktop → PC Wireless and connect to the SSID.
5. Assign IP addresses (Static or DHCP).
6. Ensure all PCs are connected to the same network.

Result:
The wireless network is successfully created using an Access Point.
